# SuguruOoki/homebrew-ccdirenv

Homebrew formula for [ccdirenv](https://github.com/SuguruOoki/ccdirenv) — direnv-style Claude Code account switching.

## Install

```sh
brew tap SuguruOoki/ccdirenv
brew install ccdirenv
```

## How releases land here

The formula `Formula/ccdirenv.rb` is generated by [cargo-dist](https://opensource.axo.dev/cargo-dist/) when each new ccdirenv version is released. After tagging `vX.Y.Z` in the source repo, GitHub Actions (`homebrew-publish.yml` in [SuguruOoki/ccdirenv](https://github.com/SuguruOoki/ccdirenv)) publishes the formula to this tap automatically.
