# SuguruOoki Homebrew Tap

Homebrew formulae for tools by [@SuguruOoki](https://github.com/SuguruOoki).

## Usage

```sh
brew tap SuguruOoki/tap
brew install <formula-name>
```

## Available formulae

| Formula | Description | Source |
|---------|-------------|--------|
| `ccdirenv` | direnv-style Claude Code account switching | [SuguruOoki/ccdirenv](https://github.com/SuguruOoki/ccdirenv) |

## How releases land here

The formula `Formula/ccdirenv.rb` is generated by [cargo-dist](https://opensource.axo.dev/cargo-dist/) when each new ccdirenv version is released. After tagging `vX.Y.Z` in the source repo, cargo-dist publishes the formula to this tap automatically.
